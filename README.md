# Infográfico Interativo - Livro das Formas de Expressão (IPHAN)

## Sobre o projeto

Este repositório contém o **front-end de um infográfico interativo** desenvolvido como parte do meu **Trabalho de Conclusão de Curso (TCC)**.

O projeto tem como objetivo apresentar de forma **visual, interativa e acessível** os bens culturais registrados no **Livro das Formas de Expressão**, pertencente ao sistema de registro de bens imateriais do IPHAN (Instituto do Patrimônio Histórico e Artístico Nacional).

A proposta do infográfico é facilitar a compreensão e a exploração desse patrimônio cultural por meio de uma interface digital interativa.

---

## Contexto

O IPHAN organiza o patrimônio cultural imaterial brasileiro em diferentes **Livros de Registro**, cada um voltado para um tipo de manifestação cultural.

Este projeto foca inicialmente no:

* **Livro das Formas de Expressão**

Ele reúne manifestações culturais como:

* Literatura de Cordel
* Roda de Capoeira
* Matrizes do Samba no Rio de Janeiro
* entre outros bens culturais registrados.

---

## Escopo da primeira versão

A **primeira versão do infográfico** contempla **apenas o Livro das Formas de Expressão**, apresentando seus bens culturais em uma interface baseada em:

* navegação por seções
* cards informativos
* sobreposições com detalhes sobre cada bem cultural
* elementos visuais interativos

Essa versão tem como foco **validar o modelo de visualização e interação proposto pelo projeto**.

---

## Possibilidades de expansão

Embora esta versão inicial apresente apenas o **Livro das Formas de Expressão**, a estrutura do projeto foi pensada para permitir futuras expansões.

Versões futuras poderão incluir também outros livros de registro do IPHAN, como por exemplo:

* Livro dos Saberes
* Livro das Celebrações
* Livro dos Lugares

Dessa forma, o infográfico poderá evoluir para uma **plataforma mais ampla de visualização do patrimônio cultural imaterial brasileiro**.

---

## Objetivos do projeto

* Tornar o patrimônio cultural imaterial **mais acessível ao público**
* Utilizar **design interativo e visualização da informação**
* Explorar **formas digitais de difusão cultural**
* Criar uma estrutura escalável que permita incluir outros livros de registro futuramente
